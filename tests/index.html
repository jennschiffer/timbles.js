Sorting tests for timbles

Timbles is mainly concerned with sorting your tables, so these tests are obviously pretty important.

There are a bunch of different ways that tables can be timblified, and all of them need to pass the same tests that verify correct instantiation, data content and sorting.

  • Plain html table
  • Plain html table with multi-row header
  • Table created from local array data
  • Table created from JSON file
  • Paginated table created from JSON file

Non-sorting tests for timbles

  • Tests for pagination of tables
  • Pagination tests where sorting and pagination are set up post-timblification
  • Miscellaneous other tests of timbles.
